1: Vad är en regel i CSS och vad består den av? Svar: Regel i CSS används för att styla HTML och består av två delar selektor och deklarationblock. Selektor avgör vilka element ska omfattas av CSS-regeln medan deklarationblock bestämmer vilka egenskaper och värde ska tillämpas på HTML-element.

2: Vad är skillnaden mellan id och class? När bör man använda id respektive class? Svar: ID och class är två typer av attribut som används för att ge elementen en identitet för att lättare strukturerar koden på och gör koden lättförstårlig. ID används för att identifiera ett specifikt element. Endas ett element kan ha ett specifik ID. I CSS markeras ID med # Ex: #logo. Class används för att identifiera flera klasser som har samma stil och funktionalitet DVS flera element kan ha en klassnamn. I CSS markerars class med punkt EX .klassnamn .
